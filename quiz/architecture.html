Architecture :

Résumé...

En NSI, nous avons étudié le thème "Architecture matérielles et système d'exploitation". Dans ce thème, il y avait plusieurs éléments :

  • ♦ Les rôles (et les caractéristiques) des différents composants d'une machine.
    (ex: RAM, processeur, CPU, ... )
    • ♦ Comment la machine exécute une séquence d'instructions simples (avec le language assembleur).
  • ♦ Qu'est ce qu'un système d'exploitation et à quoi cela sert-il ?
